# My custom cheatsheet for my vim...

## Comand hystory

q:  edit buffer for last commands
q/  edit buffer for last searches

## Registers

When in command mode <C-R> " permit to paste. add register management

## Split windows

### Create split

:s    horizontal split
:vs   vertical split

### Moving already splitted

C-w (moving keyword) : move around
C-w <shift>(moving keyword) : change layout, for example

    * C-w K change to horizontally split
    * C-w H change to vertically (h is the left moving)

C-w r : rotate the views

### Resize split

C-w <num> +/-/>/< (depending on which panel you are)
